CSS Transform

translate (x, y)

translate

translate - зрушує елемент на нове місце, переміщуючи відносно початкового положення об'єкта вправо і вниз, використовуючи координати X і Y. Записується таким чином translate (x,y);

scale (x, y)

scale

scale- масштабує елементи, роблячи їх більшими або меншими. За замовчуванням значення вказані як одиниця (1) - це реальний розмір об'єкта. Перше значення масштабує елемент за шириною, друге - за висотою. Негативні значення відображають елемент дзеркально.

rotate (угол+deg)

rotate

rotate - повертає елементи на задану кількість градусів, негативні значення повертають елемент проти годинникової стрілки, позитивні - за годинниковою стрілкою. Записуються значення як градус повороту і deg

skew (x-угол+deg, y-угол+deg)

skew

skew - деформує (спотворює) сторони об'єкта щодо координатних осей. Якщо вказано одне значення, друге буде визначено браузером автоматично. Записуються значення як градус повороту та deg

matrix (a, c, b, d, x, y)

matrix

matrix(a, c, b, d, x, y) - дає змогу об'єднати кілька функцій 2D-трансформацій в одній. Як трансформації допустимі поворот, масштабування, нахил (деформація) і зміна положення. Властивості вказуються без одиниць виміру.

Множественные 2D трансформации

translate
scale
rotate

Так само, подібного можна домогтися, використовуючи множинні трансформації. А саме записати кілька значень в одну властивість transform.

transform-origin

transform origin

transform-origin - дає змогу змістити центр трансформації, щодо якого відбувається зміна положення/розміру/форми елемента

perspective, perspective-origin

perspective, perspective-origin

perspective, perspective-origin - встановлення глибини та зміна точки початку координат

translate3d (x, y, z)

translate3D

translate3D - за аналогією зі своїм 2D братом це значення задає переміщення елемента, але в 3D-просторі. Крім осі X і Y додається вісь Z.

scale3d (x, y, z)

scale3D

scale3D - задає тривимірне масштабування за однією з осей X, Y, Z

rotate3d (x, y, z, deg)

rotate3d

rotate3d - обертає елемент відносно трьох осей. Елемент повертається під кутом, заданим останнім параметром відносно вектора напрямку X, Y,

matrix3d (n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n)

matrix3d

matrix3d - задає тривимірне перетворення як однорідну матрицю розміром 4×4 із шістнадцятьма значеннями в стовпцях

Множинні 3D трансформації

translate3d
rotate3d
